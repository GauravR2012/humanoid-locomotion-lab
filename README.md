# Humanoid Locomotion Lab

Research-style humanoid robotics locomotion framework.

Algorithms implemented:

- Linear Inverted Pendulum Model
- Zero Moment Point control
- Capture Point stabilization
- Preview control walking
- Model Predictive Control walking
- Footstep planning
- Push recovery control
- Inverse kinematics

Includes PyBullet simulation and experiment pipelines.

Run simulation:

pip install -r requirements.txt
python humanoid/simulation/simulation_runner.py
